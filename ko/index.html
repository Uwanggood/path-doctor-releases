Windows에서 느리거나 접속되지 않는 웹사이트를 진단하세요.

Path Doctor는 DNS, 왕복 지연, route hop, HTTP/TLS 응답, VPN과 proxy 단서를 확인하고 가능한 원인을 알기 쉽게 설명합니다.

Windows용 다운로드itch.io에서 후원Ko-fi에서 후원릴리스 보기
DNSPingRouteHTTP/TLSVPN / Proxy

무엇을 확인하나요

이름 조회

DNS가 빠른지, 느린지, 건너뛰어졌는지, 실패했는지 확인합니다.

왕복 지연

현재 PC에서의 응답 시간과 packet loss를 확인합니다.

경로

느린 hop, 응답 없는 구간, 필터링된 중간 장비 단서를 봅니다.

웹 응답

웹사이트 자체가 HTTP/TLS로 응답하는지 확인합니다.

스크린샷

릴리스 상태

Windows 버전을 받을 수 있습니다. macOS 지원은 같은 업데이트 채널을 통해 준비할 예정입니다. 첫 빌드는 아직 코드 서명이 없어 Windows SmartScreen 경고가 표시될 수 있습니다.
Path Doctor 공개 바이너리 릴리스 페이지입니다. 이 저장소에는 소스 코드를 공개하지 않습니다.
